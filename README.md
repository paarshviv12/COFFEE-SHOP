A beautiful, welcoming digital menu frontend interface built for **The Coffee Bistro**—a cozy corner crafted explicitly for coffee lovers. 

The layout focuses heavily on high-quality visuals, an intuitive grid layout, and aesthetic design to present premium beverages as appealingly as possible. The menu specifically highlights a curated grid including Caffe Americano, Cold Coffee, Caffe Latte, Caramel Macchiato, Mocha, Matcha Tea Latte, and Caramel Frappuccino.

## 🌟 Features
- **Product Presentation:** Clean, visual grids detailing coffee and food selections.
- **Inviting Color Palette:** Tailored warm colors that evoke the sensory feeling and atmosphere of a cozy cafe.
- **Responsive Elements:** Adapts elegantly to fit diverse screens, ranging from a customer's phone to a large point-of-sale display terminal.

## 🛠️ Built With
- **HTML5:** Robust semantic tagging and structuring.
- **CSS3:** Styling focused heavily on flexbox alignment, typography, and polished image placement.

## 🚀 Getting Started

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/paarshviv12/COFFEE-SHOP.git
   ```
2. Navigate to the `COFFEE_SHOP` directory inside the repository and open `coffeeweb.html` in your web browser.
